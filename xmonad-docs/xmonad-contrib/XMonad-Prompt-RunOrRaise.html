XMonad.Prompt.RunOrRaise

Contents

Description

A prompt for XMonad which will run a program, open a file, or raise an already running program, depending on context.

Synopsis

  • runOrRaisePrompt :: XPConfig -> X ()
  • data RunOrRaisePrompt

Usage

  1. In your ~/.xmonad/xmonad.hs: 
 import XMonad.Prompt
 import XMonad.Prompt.RunOrRaise
  1. In your keybindings add something like: 
   , ((modm .|. controlMask, xK_x), runOrRaisePrompt defaultXPConfig)

For detailed instruction on editing the key binding see XMonad.Doc.Extending.
