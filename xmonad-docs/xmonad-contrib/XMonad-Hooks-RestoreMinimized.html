XMonad.Hooks.RestoreMinimized

Contents

Description

(Deprecated: Use XMonad.Hooks.Minimize) Lets you restore minimized windows (see XMonad.Layout.Minimize) by selecting them on a taskbar (listens for _NET_ACTIVE_WINDOW and WM_CHANGE_STATE).

Synopsis

  • data RestoreMinimized = RestoreMinimized
  • restoreMinimizedEventHook :: Event -> X All

Usage

You can use this module with the following in your ~/.xmonad/xmonad.hs:

import XMonad.Hooks.RestoreMinimized

myHandleEventHook = restoreMinimizedEventHook

main = xmonad def { handleEventHook = myHandleEventHook }
